Google App Engine (often referred to as GAE or simply App Engine) is a cloud computing platform as a service for developing and hosting web applications in Google-managed data centers. Applications are sandboxed and run across multiple servers.[2] App Engine offers automatic scaling for web applicationsâ€”as the number of requests increases for an application, App Engine automatically allocates more resources for the web application to handle the additional demand.[3] Google App Engine primarily supports Go, PHP, Java, Python, Node.js, .NET, and Ruby applications, although it can also support other languages via "custom runtimes".[4] The service is free up to a certain level of consumed resources and only in standard environment but not in flexible environment.
